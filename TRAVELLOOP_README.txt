Andhra Travelloop is a web-based tourism project that helps users explore popular places across Andhra Pradesh with details like food, culture, maps, trip plans, hotel suggestions, and voice output in English & Telugu.
It also includes a login system and a simple AI chatbot.

Problem Statement :

Tourists struggle to find accurate local information
No unified platform for food, culture, maps, and trip plans
Need for simple platform + multilingual support
   
Key Features :
City selection with images & details
Automatic map display
Multilingual speech output
Daily trip plans
Clean UI & responsive layout
Real-time destination cards


System Architecture :

The project follows a client-side web architecture where all data is stored and processed in the browser using JavaScript arrays.
APIs like Google Maps are used externally.
